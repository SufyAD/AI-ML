## Stage #1: Data Ingestion/Load Stage 
Data ingestion is the process of `extracting`, `transforming` (if needed), and loading data into a storage system (e.g., databases, data lakes, warehouses). It can be batch (processed in intervals) or streaming (real-time).
    - Load from txt
    - Load from pdf
    - Load from website
